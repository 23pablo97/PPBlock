PPGame - Desarrollado por Pablo Arancibia Barahona

Comprobar si se tiene pygame, ademas de Shapely. En caso de no contener estas librerias
se pueden instalar desde el terminal usando

pip install pygame
pip install Shapely

Para la ejecucion, se abre el archivo PPGAME.py, con el siguiente comando

>> python PPGAME.py

En caso de tener problemas de ejecucion utilizar

>> python2.7-32 PPGAME.py

Si se sigue teniendo problemas de ejecucion, porfavor contactarse con el desarrollador
